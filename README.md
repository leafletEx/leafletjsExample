# 简介
记录一些 Leaflet 与 vue 结合使用的示例。

# 本地调试
克隆仓库到本地 `git clone https://github.com/vaebe/leafletjsExample.git`

安装依赖 `pnpm i`

启动 `pnpm run docs:dev`
